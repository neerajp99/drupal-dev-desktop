INSTALLING DRUPAL WITH ACQUIA DEV DESKTOP AND ADDING THEMES AND MODULES TO IT


ACQUIA DEV DESKTOP


Acquia Dev Desktop is a free app that allows you to run and develop Drupal sites locally on your computer and optionally host them using Acquia Cloud. Use Acquia Dev Desktop to evaluate Drupal, add and test other Drupal modules, and develop sites while on a plane or away from an internet connection.


HOW TO SETUP ACQUIA DEV DESKTOP?


1. Go to https://dev.acquia.com/downloads and download Acquia Dev Desktop for your Windows or Mac.







2. Extract the downlaoded file and start the installer.






















2. After installation, the dashboard will pop-up, showing all the details of your Drupal Site.







2. Now click on the local site link and it will open your Drupal local site on the web browser and the Drupal installer will begin on the web browser.







2. The Installer will not ask for Installation Profile or Database details and will directly install Drupal.







2. Now just configure site and its done.















ADD MODULES AND THEMES







- MODULES

~ To install a module just go to localsite/admin/modules and click on Install New Module.





~ Now enter the URL of the module in the Install From URL Field and click on Install.





~ Now go back to localsite/admin/modules and click on the Administration Field and Install.






- THEMES

Download the theme and put the extracted file in the themes folder inside the Drupal Local Directory and then go to localsite/admin/appearance and click on Install and Set as Default option below the theme you want to install.

HOST DRUPAL SITE ON ACQUIA SITE


- Just Click on Host Site on the dashboard of Acquia Dev Desktop.


- It will ask for User Login, If you don't have an Acquia account then make one and login.


- Now just enter the Private key and your website will be hosted on Acquia Cloud

     ~ To generate a 4096 bit ssh key, use the command
       $ ssh-keygen -b 4096 -f username -t rsa





BENEFITS OF USING ACQUIA DEV DESKTOP


- Easy to Install

- No need to install MySQL, PHP, Apache Server. Acquia Dev Desktop will do everything for you.

- Easy to host websites with Acquia Cloud.
